Catholic Relief Services

The following 2 examples are work I oversaw that my team at CRS produced.

The first is a site that we did for CRS' Advocacy team. They were looking to develop a new site for the Catholics Confront Global Poverty advocacy campaign that CRS does in conjunction with the USCCB. In the case of this project my team was the User Experience consultant but the site was designed and built by an outside agency. We did the IA, site map and wireframes for the project. I am attaching an example of the wireframes. The final live site can be seen at: http://www.confrontglobalpoverty.org/

The second is a site we did for CRS' fund-raising team. This site was for the CRS Helping Hands program which is a community fund-raising and meal packing program where Parish's or Schools can hold a meal packing event. We did the the entire site for this project, IA, site design and custom Wordpress theme development and programing. I am attaching the IA work product. The final live site can be seen at: http://helpinghands.crs.org.
